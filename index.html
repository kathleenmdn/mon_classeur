Qu'est ce qu'une IP ?

Une IP est un protocole/une chaîne de caractère constituée de 4 groupes de chiffres allant de 0 à 254.
Adresse IP est un nom informatique qui va être donnée à un appareil connecté.

Une IP peut-être être locale, on va donc parler d'une IP LAN (=unique car elle n'est pas la même que votre voisin par exemple), lorsque qu'elle est publique (internet) on va donc dire IP WAN :

Nous pouvons donc joindre l'ordinateur X mais il ne faut pas que l'on se retrouve sur l'ordi Y c'est pour cela que les IP doivent être uniques.
