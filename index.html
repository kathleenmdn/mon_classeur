Qu'est ce qu'une IP ?

Une IP signifie Internet control et est une chaîne de caractère allant de 0 à 254.
Adresse IP est un nom informatique qui va être donnée à un appareil connecté.

Une IP peut-être être locale, on va donc parler d'une IP LAN (=unique car elle n'est pas la même que votre voisin par exemple) :
